Write a class that interacts with the public Star Wars API (SWAPI) for "people" located at https://swapi.dev/api/people.

## Acceptance Criteria

- retrieve ALL people
- sort people by:
1. number of films (descending)
2. name (ascending)
- return all sorted people as objects like so: { name: String, filmCount: Number }

## Looking for:

- Proper syntax -- classes, methods, types, etc
- Use async/await for all API methods
- Working, testable code
- Collaboration, "thinking out loud"
